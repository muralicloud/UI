1.The centralized system works on a Client Server Relationship. The repository is located at one place and provides access to many clients. Whereas in Distributed System every user has a local of the repository in addition to the central repo on the server side.

2. The 2 types of Centralized version are SVN and Perforce. And the 2 Distributed System are Git and Mercurial.

3. The advantage of using GIT over other VCS are -
        a. Performance
	b. Security
        c. Branching Model
	d. Staging area


4.There are 3 stages -
      a. Modified
      b. Staged
      c. Committed
